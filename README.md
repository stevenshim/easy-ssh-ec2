# Let's ssh ec2 easily.

## Prerequisites
* Locate your ssh `key pair` on `~/.ssh/id_rsa`.
* `awscli` must installed.
* `access key pair` or `Role` has permission to describe instances.
<br><br>

## Update EC2 List
```bash
$ ec2.sh
```
